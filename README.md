# **ATM CLI application**

## App features
- Change Pin (you can create a default pin in your code and then change it in the function)
- Check Account balance
- Withdraw funds
- Deposit funds
- Cancel/Exit (selecting this option should exit the program)

## Usage
```bash
go run .
or
go run main
```
